# Purdue Data Science & AI

Personal coursework and project portfolio from Purdue's Post Graduate Program in Data Science, Machine Learning, and Artificial Intelligence, plus related self-study projects.

This repository contains only my own original work — completed notebooks, scripts, and write-ups I authored or submitted. Instructor slide decks, starter/template notebooks, answer keys, licensed datasets, IBM/Coursera-authored lab notebooks, and other paid course materials are intentionally excluded.

## Structure

- **PC DS - Machine Learning/** — Machine learning coursework: course-end project (`cep2/`), the employee turnover analytics project, and in-class practice notebooks organized by session date.
- **PC DS - Deep Learning Specialization/** — Deep learning coursework: course-end project on loan and port-operations analysis (`CEP/`), a YOLO object detection notebook, and in-class practice notebooks organized by session date.
- **PC DS - Applied Data Science with Python/** — Course-end projects analyzing marketing and sales data.
- **PC DS - Database Management using SQL (wk2-4)/** — SQL coursework, including the Air Cargo course-end project (analysis write-up, PDF report, and SQL script).
- **PC DS - Essentials of Generative AI, Prompt Engineering & ChatGPT/** — Prompt engineering notes and course-end project.
- **PC DS - GEnerative AI for Data Professionals/** — Fraud detection and smartwatch course-end projects.
- **IBM/** — Completed notebook quizzes from IBM Skills Network Python modules (lab notebooks with IBM's own instructional content excluded).
- **personal expense tracker project/** — Self-directed project building a personal expense tracker.
- **files/** — Miscellaneous original exercises.

## Notes

- Large raw datasets used in coursework are not included in this repository to keep it lightweight; notebooks reference the data sources used at the time.
- Some folders retain their original working names (e.g. session dates, project shorthand) as they were organized during the program.
